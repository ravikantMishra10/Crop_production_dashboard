🌾 Crop Production Dashboard
📌 Project Overview

The Crop Production Dashboard is an interactive Excel-based data analytics project designed to analyze agricultural production data. The dashboard provides clear insights into crop yield, seasonal trends, regional production, and performance comparisons using dynamic visualizations.

It helps users understand agricultural patterns and supports data-driven decision-making for farming and planning.

🎯 Objectives

Analyze crop production across different regions

Compare yearly and seasonal crop yield trends

Identify high-performing crops

Monitor production performance using KPIs

Provide interactive filtering for better analysis

🛠️ Tools & Technologies

Microsoft Excel

Pivot Tables

Pivot Charts

Slicers & Timelines

Data Cleaning & Transformation

Dashboard Design & Visualization

📊 Dashboard Features

✅ Interactive filters using slicers
✅ Dynamic charts and visual reports
✅ KPI indicators for production analysis
✅ Region-wise and crop-wise comparison
✅ User-friendly agricultural theme design

📂 Dataset

The dataset contains agricultural production data including:

Year

Seasonal Information
